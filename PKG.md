# 3rd Party Libraries

This document lists the 3rd party libraries used in Gobungle, classified by their primary function.

## Terminal UI & Control
- **tcell**: A comprehensive Go package for building terminal-based user interfaces.
- **golang.org/x/term**: Provides terminal-specific functions and state management.

## Text & Character Handling
- **uniseg**: Handles Unicode text segmentation for grapheme clusters, words, and sentences.
- **golang.org/x/text**: A collection of packages for text processing, including Unicode support and localization.
- **encoding**: Provides various character encodings, used by tcell for terminal compatibility.

## Graphics & Visuals
- **go-colorful**: A library for advanced color manipulation, blending, and color space conversions.

## System Primitives
- **golang.org/x/sys**: Low-level, platform-specific operating system primitives used for terminal control.
